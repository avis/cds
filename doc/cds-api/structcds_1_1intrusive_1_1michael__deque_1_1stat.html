Detailed Description

template<typename Counter = cds::atomicity::event_counter>
struct cds::intrusive::michael_deque::stat< Counter >

Deque internal statistics. May be used for debugging or profiling.

Template argument Counter defines type of counter. Default is cds::atomics::event_counter. You may use other counter type like as cds::atomics::item_counter, or even integral type, for example, int.

The class extends intrusive::deque_stat interface for MichaelDeque.
