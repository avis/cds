Detailed Description

template<typename T>
class cds::details::Comparator< T >

Comparator.

Comparator is a functor (a class with binary operator() ) that compares two values. The comparator is based on std::less<T> functor and returns the result of comparing of two values:

  • -1 if a < b
  • 0 if a == b
  • 1 if a > b
